webot v0.1 
本模型使用了gpt-4o，未开通微信支付的微信用户无法使用
功能包含：
1. 微信聊天机器人：通过 ItChat 监听微信消息并进行自动回复。
2. Azure OpenAI 集成：调用 Azure OpenAI 的 GPT 模型生成回复。
3. 用户风格学习：根据用户的聊天历史分析其风格，并在回复中模仿。
4. Flask 控制面板：提供一个网页界面，用于管理配置和监控。
5. 数据持久化：将用户风格和聊天记录存储到本地文件或数据库中。

使用方式：
1. 点击webot.exe运行。
2. 扫描二维码，如果命令行中的二维码无法扫描，可以扫描exe文件运行后，在其下生成QR.png。
3. 等待网页自动登录
4. 打开右上角的 打开控制面板，选择想要模仿的用户，然后开启启用自动回复
5. 第一次运行后会生成三个文件：
	chat history.db 用于储存聊天记录
	QR.png 如上，用于登录微信
	user_styles.json 用于储存用户风格
删除三个文件等于对程序初始化


其他：
1. 微信中不会回复自己，在命令行和控制面板中可以看见回复自己的内容
2. 在聊天对象停止回复5-10s内，会把之前对方的聊天内容整合然后进行回复
3. 一直回复gg，说明有报错
4. 内置一个Azure OpenAI api，每分钟令牌数 100,000, 每分钟请求数 600
5. 不能回复图片，表情包和群消息
